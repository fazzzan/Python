# деление по остатку 
![](../_pictures/image_20250402111654.png)

деление по остатку можно использовать при анализе арифметических рядов:
- 8, 15, 22, 29... - дают при %7 = 1
- 10, 17, 24, 31... - дают при %7 = 1

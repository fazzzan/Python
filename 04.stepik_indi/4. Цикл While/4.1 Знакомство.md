## Удаление из списка
```python
a = [1, 3, 4, 3, 6, 3]
while 3 in a:
    a.remove(3)
print(a)  # [1, 4, 6]
```
![](Pasted%20image%2020250519134048.png)

Тут основная прелесть - если проверить на несуществующий элемент списка, то ошибки при попытке удаления несуществующего элемента - не будет.

–í–∞—à–∞ –∑–∞–¥–∞—á–∞ ‚Äî —Å–æ—Ö—Ä–∞–Ω–∏—Ç—å –Ω–∞–∏–º–µ–Ω—å—à–µ–µ –∏–∑ —ç—Ç–∏—Ö —á–∏—Å–µ–ª –≤ –ø–µ—Ä–µ–º–µ–Ω–Ω—É—é¬† `minimum`, –∞ –Ω–∞–∏–±–æ–ª—å—à–µ–µ - –≤ `maximum`, –ò—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –Ω—É–∂–Ω–æ, –∫–æ–Ω–µ—á–Ω–æ –∂–µ, —Ç–µ—Ä–Ω–∞—Ä–Ω—ã–π –æ–ø–µ—Ä–∞—Ç–æ—Ä
–í –∫–∞—á–µ—Å—Ç–≤–µ –æ—Ç–≤–µ—Ç–∞ –≤—ã–≤–µ–¥–∏—Ç–µ —á–µ—Ä–µ–∑ –ø—Ä–æ–±–µ–ª —Å–ø–µ—Ä–≤–∞¬†`minimum`¬†, –∞ –ø–æ—Ç–æ–º¬†`maximum`
```python

```
